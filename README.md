# suport-color

ANSI color support detection for terminal applications

## Overview

Detects terminal color support level — no color, 16 colors, 256 colors, or true color. Works across all major operating systems.

## Features

- Cross-platform color detection
- True color (16.7M) support detection
- NO_COLOR and FORCE_COLOR support
- CI environment detection
- 0 dependencies

## Installation

```bash
npm install suport-color
```

## Quick Start

```javascript
npm import suport-color
```
